### Detailed Description

VirusTotal Domain Monitor is a developer tool for webmasters who need to keep track of their domains' security reputation.

**How it works**
Add your domains to a watchlist, enter your free VirusTotal API key, and the extension automatically checks each domain on a schedule you choose. A color-coded badge on the extension icon shows the status of the current site at a glance.

**Key features**
- Watchlist monitoring — add domains for automatic scheduled checks (12h to 7 days)
- Badge for every site — see reputation status (clean, suspicious, malicious) on the extension icon as you browse
- Bulk add — paste a list of domains, URLs, or mixed text to add many at once
- Smart Check — automatically requests a VT rescan when data is stale, configurable via Rescan Policy
- Dispute false positives — one-click access to 60+ vendor dispute forms with ready-made templates and AI prompts
- Per-vendor tracking — mark each vendor as Not disputed / Disputed / Resolved
- Pause mode — pause all checks for 1 hour to save API quota
- IDN support — Unicode display with punycode on hover
- Dark and light theme with auto-detection
- Works on Chrome, Edge, and Firefox
- Fully local — no analytics, no tracking, data stays in your browser

**For webmasters**
This extension is designed for people who manage multiple websites and need to know if any of their domains get flagged by antivirus vendors. Instead of checking VirusTotal manually, the extension does it in the background and alerts you when something changes.

**Privacy**
The extension only communicates with the official VirusTotal API using your own API key. No data is sent anywhere else. No analytics, no telemetry, no third-party services. See our full privacy policy for details.

**API key**
You need a free VirusTotal API key to use this extension. Create an account at virustotal.com and copy your Public API key from the API section. The free tier allows 500 lookups per day — enough for most webmasters.
